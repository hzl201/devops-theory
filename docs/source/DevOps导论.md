## **软件开发的本质困难**

- 复杂性
- 不可见性
- 可变性
- 一致性


## **网络化和服务化**

> 软件应用特征
>> 功能更复杂，规模更大
>> 用户数量急剧增加
>> 快速演化和需求不确定
>> 分发方式的变化（SaaS）

## **迭代式软件开发方式**

> 迭代式：大型软件系统的开发过程也是一个逐步学习和交流的过程，软件系统的交付不是一次完成，而是通过多个迭代周期，逐步来完成交付。
